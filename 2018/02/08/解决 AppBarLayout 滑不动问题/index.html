解决AppBarLayout滑不动的问题

使用Android 5.0中新添加的AppBarLayout可以比较方便的实现如下的效果

Layout布局文件

在布局文件的定义中：
用CoordinatorLayout包住AppBarLayout,滚动 view 是 RecyclerView。

这样就实现了一个简单的 RecyclerView 与 appbarLayout 的级联滚动效果了，但是实现完成之后有一个问题就是，当我 只要在 滚动 view 区域内触摸过，再去 AppBarLayout 区域进行触摸滑动时 appbarLayout 区域内容就滑动不了，而只可以在滚动 view 区域内滑动。

我自己实现的解决办法是在 AppBarLayout 里添加 ScrollView 就可以解决。但引出一个 bug 就是有自定义的下拉刷新失效了
