Golang 在 Mac OS 上的环境配置

网上有很多安装方法，比如通过 homebrew 来安装。

直接下载的安装包

安装包下载地址
地址： https://golang.org/dl/

Homerbrew 安装参考:http://brew.sh/.
使用命令行直接安装Golang:

验证安装成功
在命令行输入 go version，获取到go的version，则代表安装成功。我的验证结果如下：

go version

配置环境变量

查看go 的环境变量设置的命令

go env

一般来说，需要为go添加三个主要的环境变量，分别是GOROOT, GOPATH以及go命令的路径GOBIN，通过安装包点击安装的go的位置默认为/usr/local/go，因此需要将/usr/local/go/bin添加到PATH里，而GOROOT便是安装目录/usr/local/go，所以GOROOT变量默认已经设置好。GOPATH一般是自定义目录，日常开发的根目录，用于存放你将要源码位置，如$HOME/go等，修改环境变量可通过修改当前用户目录下的配置文件来修改，GOBIN:是GOPATH下的bin目录。

简单说，需要设置的环境变量包括:GOPATH ,GOBIN 以及把GOBIN加入到PATH中,GOROOT变量默认已经设置好。。

再看下go的环境变量:

go env

环境变量设置完成!

卸载

找到安装目录/usr/local/go，删除 go 目录及删除环境变量即可。
